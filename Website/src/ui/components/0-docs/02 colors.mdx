# Colors

Colors are defined as variables in Sass in the `/styles/1-settings/_settings.colors.scss` file.
The colors file also contains all color definitions for the underlying components.

## Named colors

The named colours are the named brand colors for the site. The colours are typically used for to highlight and accentuate elements on the site.
